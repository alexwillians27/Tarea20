# AI-Assisted Task Manager 📝

**Author:** Alejandro Mancheño  
**Course:** 1º DAW  

## About the Project
This is a command-line Task Manager application built with Java. It was developed as part of **Task 20** to demonstrate two main goals:
1. **AI Pair Programming:** Using Generative AI as a collaborative tool to help write logic, structure the code, and debug errors.
2. **Git Mastery:** Managing version control, branching (using a `feature-update` branch), and deployment entirely through the Git Command Line Interface (CLI).

## Features
* **Add Tasks:** Easily add new tasks to your to-do list.
* **List Tasks:** View all pending tasks with their corresponding index numbers.
* **Delete Tasks:** Remove completed tasks from the list safely.
* **Input Validation:** The program handles user input errors gracefully (e.g., typing letters instead of numbers) using a `try-catch` block.

## Technologies Used
* **Java:** Core application logic, `ArrayList` for data storage, and `Scanner` for input.
* **Git & GitHub:** Local version control, branching, merging, and remote repository hosting.
* **GitHub Pages:** Project deployment and documentation hosting.
* **Generative AI:** Assistant for code structure, debugging, and Git CLI troubleshooting.
